# Carpeta de Audio

Coloca aquí el archivo de audio que quieres que se reproduzca durante la navegación.

**Formato recomendado:** MP3, OGG o WAV

**Nombre del archivo:** El archivo debe llamarse `background.mp3` (o cambiar la extensión según el formato que uses)

Si usas otro nombre, actualiza la referencia en el archivo `index.html` en la línea del elemento `<audio>`.
