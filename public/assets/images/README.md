# Carpeta de Imágenes

Coloca aquí las fotos que quieres mostrar en la galería.

**Formatos soportados:** JPG, PNG, WEBP

**Recomendaciones:**
- Usa imágenes de buena calidad
- Optimiza las imágenes para web (tamaño recomendado: máximo 2MB por imagen)
- Las imágenes se mostrarán automáticamente en la galería

**Nota:** Si quieres que las imágenes se carguen automáticamente, asegúrate de que los nombres de archivo sean descriptivos. El script buscará archivos con extensiones .jpg, .jpeg, .png y .webp en esta carpeta.
